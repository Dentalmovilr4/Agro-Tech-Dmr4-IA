# 🌾 Agro-Tech-Dmr4-IA
> **Optimización Energética Híbrida para el Sector Arrocero**

Este proyecto es una solución digital avanzada desarrollada por **Dentalmovilr4** para transformar los residuos agrícolas en independencia energética.

## 🚀 Características Principales
- **🔥 Gestión de Biomasa:** Cálculo preciso de generación eléctrica mediante la combustión de cascarilla de arroz.
- **☀️ Simulación Solar:** Dimensionamiento de paneles fotovoltaicos optimizados para la radiación del Cesar, Colombia.
- **💰 Análisis Financiero:** Cálculo automático de Retorno de Inversión (ROI) y ahorro mensual en COP.
- **📐 Guía Técnica:** Recomendaciones de inclinación (10°-12°) y orientación (SUR) para máxima eficiencia.

## 🛠️ Tecnologías Usadas
- **Python 3.10+** (Motor de cálculo)
- **Streamlit** (Interfaz de usuario dinámica)
- **Pandas** (Procesamiento de datos financieros)
- **GitHub Actions** (Auditoría mediante IA DMR4)

## 📍 Ubicación Técnica Optimizada
El sistema está configurado por defecto para las condiciones climáticas de **El Copey, Cesar**, aprovechando un promedio de 5.8 Horas Sol Pico (HSP).
